# Booking Template

Static hotel and booking website template built with Bootstrap assets, SCSS source files, and multi-page HTML views.

## Scope

- Hotel-style landing page and supporting pages for rooms, booking, about, and contact.
- Bootstrap-based responsive layout.
- SCSS source files and compiled CSS assets.
- Carousel, animation, date/time picker, and UI utility libraries bundled for static hosting.

## Tech Stack

- HTML
- SCSS/CSS
- Bootstrap
- JavaScript
- jQuery plugins

## Project Structure

```text
.
|-- index.html
|-- about.html
|-- booking.html
|-- contact.html
|-- room.html
|-- css/
|-- js/
|-- img/
|-- lib/
`-- scss/
```

## Run Locally

Open `index.html` directly in a browser or serve the folder with any static file server.
